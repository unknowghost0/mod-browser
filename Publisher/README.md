# Publisher Helpers

The community repo flow changed.

Use one of these:

- the in-game `CREATE MOD` view
- `New-CommunityEntry.ps1`

Do not use the old one-file `submission.json` or manual catalog merge flow anymore.

Modern community entries live in category folders:

- `Mods/<ProjectName>/`
- `TexturePacks/<ProjectName>/`
- `WeaponAddons/<ProjectName>/`

Each folder should contain:

- `mod.json`
- `README.md`
- `preview.png` or `preview.gif`
